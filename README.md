# 安念善終 — 一條龍善終服務 Website & Mobile PWA

陪你安放思念，從容送別所愛。

## 📁 專案結構

```
an-nian/
├── index.html              # Landing Page（主站）
├── styles.css              # CSS 樣式系統（含深色模式 + RWD）
├── script.js               # 互動邏輯（導航 / 主題 / WhatsApp CTA）
├── manifest.json           # PWA 應用程式描述
├── assets/
│   └── logo.svg            # 蓮花瓣 monoline 標誌
├── extensions/
│   └── brand-assets.txt    # WhatsApp 文案、名片設計、Google Ads 文案
├── sw.js                   # Service Worker（離線緩存）
├── vercel.json             # Vercel 部署配置
└── README.md               # 說明文件
```
